# 🏥 HealthStore API

A secure RESTful Web API built with ASP.NET Core 8.0 for managing patients and vitals, with JWT-based authentication and role-based access control using ASP.NET Identity.

---

## 📦 Tech Stack

- **.NET 8.0**
- **ASP.NET Core Web API**
- **Entity Framework Core**
- **ASP.NET Identity** (IdentityUser, IdentityRole)
- **JWT Authentication**
- **AutoMapper**
- **SQLite / SQL Server**
- **Swagger (OpenAPI)**

---

## 🚀 Features

- ✅ JWT-based authentication (Login/Register)
- ✅ Identity integration with custom role support
- ✅ Role-based authorization (Admin, Doctor, Patient)
- ✅ CRUD APIs for:
  - Patients
  - Vitals
- ✅ AutoMapper for clean DTO mapping
- ✅ EF Core seeding for roles and initial data
- ✅ Token generation for email/password reset
- ✅ Swagger UI with JWT token support
